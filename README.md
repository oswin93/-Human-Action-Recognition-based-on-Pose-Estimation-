# -Human-Action-Recognition-based-on-Pose-Estimation-
The following are the major tasks performed: 1. Implementation of Convolutional Neural Network based pose estimation for body landmark detection 2. Implementation of pose features-based action recognition and its improvement using graphical feature representation and data augmentation of body landmarks 3. Preparation and preprocessing of image datasets 4. Fine tuning and improvement of the action recognition model with better feature representation and data augmentation 5. Development, error analysis and deep learning model improvement
